# Allo Inventory Reservation System

A Next.js inventory reservation system built for multi-warehouse ecommerce inventory management.

## Features

- Product inventory tracking
- Warehouse-based stock management
- Reservation-based checkout system
- Reservation confirmation and cancellation
- Automatic expiry cleanup
- Transaction-safe reservation handling
- Live countdown timer

## Tech Stack

- Next.js
- TypeScript
- Prisma ORM
- Supabase PostgreSQL
- Tailwind CSS

## Running Locally

```bash
npm install
npm run dev
